# ☕ Polina's Cafe - WPF Billing System

Сучасний десктопний додаток із затишним графічним інтерфейсом на платформі **WPF (.NET / C#)**, розроблений для автоматизації обслуговування клієнтів та формування замовлень у кав'ярні **Polina's Cafe**.

---

## 🎨 Особливості дизайну
* **Затишна тема кав'ярні:** Інтерфейс виконано в теплих тонах еспресо, молочної пінки та карамелі.
* **Сучасний UI:** М'які закруглення блоків, елегантні тіні та зрозуміле групування елементів управління.
* **Інтерактивний чек:** Графічне відображення чека (`Guest Receipt`) у стилі меню з миттєвим оновленням підсумків.

---

## 🚀 Основний функціонал

* **1. Додавання позицій (Add Item):**
  * Введення опису страви (обмеження `[3-20]` символів).
  * Введення ціни (значення `> 0`).
  * Ліміт замовлення — максимум **5 позицій** у чеку.

* **2. Видалення позицій (Remove Item):**
  * Зручний вибір страви для видалення зі спадного списку `ComboBox`.
  * Автоматичний зсув елементів списку без порожніх проміжків.

* **3. Налаштування чайових (Add Tip):**
  * Вибір режиму за допомогою `RadioButton`: відсоток (%), fixed сума ($) або без чайових.
  * Динамічне блокування/розблокування полів введення залежно від обраного режиму.

* **4. Автоматичні розрахунки:**
  * **Net Total:** Чиста сума страв.
  * **Tip Amount:** Обчислена сума чайових.
  * **GST Amount:** Податок у розмірі 5%.
  * **Total Amount:** Фінальна сума до сплати.

* **5. Робота з файлами (Save / Load CSV):**
  * Експорт сформованого чека у файл `.csv` у корінь проєкту.
  * Імпорт даних із збереженого CSV-файлу з повною валідацією вмісту.

## 🛡 Захисне програмування (Defensive Programming)
* Програма повністю захищена від аварійних вильотів: усі некоректні введення (букви замість чисел, задовгі назви, порожні значення) перехоплюються.
* Користувач отримує зрозумілі попередження через інформаційні вікна `MessageBox`.
